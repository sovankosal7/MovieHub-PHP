# Movie Hub - Full Stack PHP Project

A dynamic movie management system featuring a custom-built PHP backend and MySQL database.

## 🚀 Features
*   **User Authentication:** Secure login and registration.
*   **CRUD Operations:** Add, Edit, and Delete movies from the database.
*   **Search:** Live search functionality for movie titles.
*   **Responsive Design:** Mobile-friendly UI (Integrated via Template).

## 🛠️ Tech Stack
*   **Backend:** PHP
*   **Database:** MySQL
*   **Frontend:** HTML5, CSS3, JavaScript (Template by [HARI GOVIND])
*   **Environment:** XAMPP / Apache

## 📥 Installation & Setup
1. Clone this repository into your `htdocs` folder.
2. Open **phpMyAdmin** and create a new database named `online_movie`.
3. Import the SQL file located at `/database/online_movie.sql`.
4. Update `db.php` with your local database credentials if necessary.
5. Visit `localhost/your-project-folder` in your browser.
